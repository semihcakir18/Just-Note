# Just Note

A simple Chrome extension that lets you take notes per URL. Each page gets its own notes.

## Features

- Notes are saved per exact URL (`example.com/1` and `example.com/2` are separate)
- Auto-saves as you type
- Works on any page including PDFs
- View all saved notes in one place
- Uses browser local storage (no accounts, no servers)

## Install

1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this folder

## Usage

Click the extension icon on any page to start writing notes. They save automatically.
